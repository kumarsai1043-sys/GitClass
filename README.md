VCS: Version Control System
    : It is used to track and manage the versions of source code

Types of VCS:
    1. Localized VCS : It is the type of VCS that tracks the version of source code locally ( i.e., from our local machine )
                      : Tracking and Back tracking of a particular version of source code is possible
                      : Risk of Data loss
                      : No collaboration between system
